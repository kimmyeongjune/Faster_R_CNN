**model의 구조는 현재 제안되어있는 파라미터들이 최적이라고 판단하여, batch_size및 lr만을 조절하여 학습을 진행하였다. 초기에는 높은 lr로 수렴에 집중하였고, 이후에는 낮은 lr로 global optima를 찾으려고 노력하였습니다.
